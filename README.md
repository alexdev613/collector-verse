# CollectorVerse

Aplicação web para gerenciamento de coleções pessoais, incluindo action figures e livros.

## 🚀 Tecnologias
- React
- TypeScript
- Vite
- TailwindCSS

## ✨ Funcionalidades
- Dashboard de personagens
- Cards interativos com navegação
- Estrutura escalável para múltiplas coleções
